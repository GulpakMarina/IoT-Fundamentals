## Getting Started

### Prerequisites

- [Arduino IDE](https://www.arduino.cc/en/software) 1.8+ or Arduino IDE 2.x
- Appropriate board support packages installed (e.g., ESP8266/ESP32 via Boards Manager)
- Required hardware components listed in each project

### Running a Project

1. Navigate to the project folder.
2. Open the `.ino` (or `.c`) file in the Arduino IDE.
3. Install any required libraries via **Sketch → Include Library → Manage Libraries**.
4. Select your board and port under **Tools**.
5. Upload the sketch to your device.
